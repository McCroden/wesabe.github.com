Wesabe.com is no more...

Wesabe, the personal finance manager, shut down in July of 2010. We kept the discussion forums up for a couple more years, since they were relatively low maintenance, but they eventually started attracting more spam than not, so we decided it to shut that down as well.

...but the code lives on

Most of the the code behind Wesabe is now in the public domain, on GitHub.

Thanks!
